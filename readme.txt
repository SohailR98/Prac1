Prac1 - Python Practice Repository

Welcome to the "prac1" repository! This repository is dedicated to practicing fundamental Python concepts. Whether you're a beginner looking to learn Python or someone seeking to refresh their skills, this repository aims to provide small Python scripts covering various basic topics.

Purpose
The primary goal of this repository is to help users become familiar with Python syntax and fundamental programming concepts. By practicing small code snippets, you can reinforce your understanding of topics such as if-else statements, loops, functions, and more.

Contents
Currently, this repository contains Python scripts covering the following topics:

If-else statements
Print statements
Loops (including for and while loops)
and more topics to come

Usage
To explore the code samples in this repository, simply navigate to the respective Python files for each topic. Each script is self-contained and includes comments for clarity. Feel free to modify the code, experiment with different inputs, and expand upon the examples provided.

Contributions
Contributions to this repository are welcome! If you'd like to contribute additional code snippets covering new Python topics or improve existing ones, please feel free to submit a pull request.
